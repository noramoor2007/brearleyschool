HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. HTML describes the structure of a web page using various elements and tags.

Everyone has an app called TextEdit on their computers.

  1. Open TextEdit.
  2. Click "Untitled" at the top and replace the name of the file.
  3. Go to the bar at the top and go to "Format" and click "Make Plain Text".
  4. Locate the file in "Finder".
  5. Control-click the file, to reveal the following:
  6. Click "Rename" and after the name you entered, add ".html" to the name. You will get this pop-up:
  7. Click "Use .html" and return to "TextEdit".
    8. Now you can start coding away!
